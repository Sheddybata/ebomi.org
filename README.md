# EBOMI Website

Evangelical Biblical Outreach Ministries International - A modern, responsive website built with Next.js 14+ (App Router), Tailwind CSS, and Lucide Icons.

## Brand Identity

- **Vibe**: Radical, Prophetic, and Authoritative
- **Colors**: 
  - Deep Navy Blue (#0A1929)
  - Gold Accents (#D4AF37)
  - Clean White Space

## Features

### Pages

1. **Home** - Hero slideshow, upcoming events, ministry arms, books, recent programs, gallery preview
2. **Gallery** - Photo gallery with pagination, automatically loads images from public/gallery
3. **Give** - Floating button with modal for account details
4. **Upcoming Programs** - Event listings
5. **Download Centre** - Sermons and resources
6. **Branches** - Searchable directory of church locations
7. **Join Us** - Prayer army information and registration
8. **Library** - Prophet's books collection

### Key Features

- **Multi-language Support** - Multiple languages with flag icons
- **Global Search** - Site-wide search functionality
- **Pull-to-Refresh** - Mobile-friendly refresh
- **Back to Top** - Smooth scroll to top button
- **Countdown Timer** - Event countdown for Global Revival Congress
- **Scroll Snap** - Smooth horizontal scrolling sections
- **SEO Optimized** - Meta tags, structured data, sitemap, robots.txt
- **Performance Optimized** - Lazy loading, code splitting, image optimization

## Tech Stack

- Next.js 14+ (App Router)
- TypeScript
- Tailwind CSS
- Lucide Icons
- React 18
- react-country-flag

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Build

```bash
npm run build
```

## Deployment

This project is configured for deployment on Vercel. Simply connect your GitHub repository to Vercel and it will automatically deploy.
